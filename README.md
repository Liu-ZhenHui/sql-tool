# SQL 工具

一个简洁的 SQL 文件管理器和编辑器。

## 功能

- 文件管理：创建、编辑、删除 SQL 文件
- 文件夹支持：创建文件夹，移动文件
- 自动补全：SQL 关键字、表名、字段名
- 格式化：自动格式化 SQL 语句
- 本地同步：连接本地文件夹，自动同步
- 定时保存：每30秒自动保存，关闭时保存

## 运行方式

### 方式1：直接运行（推荐）
```
通过打包.bat打包后会有一个dist文件夹，里面有sql_tool.exe
双击 sql_tool.exe
```

### 方式2：Python 运行
```
python sql_tool.py
```

### 方式3：浏览器打开
```
用浏览器打开 index.html
```
